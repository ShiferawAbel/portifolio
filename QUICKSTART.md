## 🌐 How to View & Deploy

### View Locally (Quick Preview)
1. Open `index.html` in your web browser
2. Or use Python: `python3 -m http.server 8000`
3. Then visit: http://localhost:8000

### Deploy to Web
1. Upload all files to your web hosting (Vercel, Netlify, GitHub Pages, etc.)
2. Set `index.html` as your homepage
3. Done! Your portfolio is live
